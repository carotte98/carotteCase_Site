<h1>Projet CodeCase</h1>
<br>
<h2>Ce projet est un site web permettant aux utilisateurs de :</h2>
<ul>
  <li>Partager du code, à travers des posts</li>
  <li>Commenter des posts</li>
  <li>Liker des posts</li>
  <li>Taguer des posts</li>
  <li>Enregistrer des posts dans son espace personnel</li>
  <li>Générer un lien du post</li>
  <li>Rechercher, via une barre de recherche, par mots clés ou tag</li>
</ul>
<p>L'utilisateur pourra se connecter grâce à un user (qui est son adresse mail en premier lieu, puis deviendra son pseudo après la confirmation de son adresse mail) et à son mot de passe.</p>
<p>Il pourra ainsi accéder à son espace personnel et y retrouver ses informations personnelles qu'il pourra modifier ainsi que les posts qu'il a enregistrer</p>
<br>
<h2>Les technos utilisées :</h2>
<ul>
  <li>Installer l'IDE IntelliJ</li>
  <li>JDK 25</li>
  <li>Installer SonarQube pour les tests</li>
  <li>Utilisation du moteur de rendu Thymeleaf pour les pages web</li>
  <li>Utilisation de l´environnement Spring</li>
  <li>Créer l´environnement du projet avec Spring Initializr (https://start.spring.io/)</li>
  <li>Utiliser Maven pour les dépendances</li>
  <li>Utiliser Tomcat pour le déploiement</li>
  <li>Utilisation de MySQL pour la BDD</li>
  <li>Environnement de test avec Postman</li>
</ul>
